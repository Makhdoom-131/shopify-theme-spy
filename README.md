# 🔍 Shopify Theme Spy

Instantly reveal the exact theme, marketing apps, and payment gateways of any Shopify store.

## ✨ Features

- **Theme Extraction:** Uncovers hidden theme names and Theme IDs.
- **App Fingerprinting:** Detects Klaviyo, Loox, Judge.me, ReCharge, and more.
- **Payment Gateway Audit:** Finds Apple Pay, Shop Pay, PayPal, Google Pay.
- **Zero API Keys:** Uses raw HTTP string matching via Cloudflare Workers.

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Cloudflare Workers (Serverless)
- **Architecture:** GitHub Pages → Cloudflare Worker → Target Website
